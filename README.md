# Raytrace Software Renderer

This is a simple raytracer implemented in C++. It only supports spheres. It also calculates basic diffuse, ambient lighting. I added a fresnel effect to the spheres, which is a nice touch. I also ended up parallelizing the raytracing to speed it up.
